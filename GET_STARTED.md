# Getting Started

I haven't made any firmware for it so it is up to you to make yourself. Just treat it like a normal RP2350A board with a spi display and battery management.

### SPI Pins:
- SCLK GPIO 2
- SDI GPIO 3
- D/C GPIO 4
- CS GPIO 5
- RES GPIO 6
- BUSY GPIO 7

## How to install the battery:
- Just plug it into the top connector on the board

## How to install the display:
1. Lift the little arm on the fpc connector
2. Insert the display's fpc cable with the metal contacts facing down
3. use the switch to decide which resistance to use
4. Press the arm back down to secure the cable in place
5. Done!
